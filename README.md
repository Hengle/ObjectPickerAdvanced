# ObjectPickerAdvanced
Unity 对象选择器

# 原因
默认的对象选择器，无法设置指定的路径来选择对象。所以仿照自带的对象选择器来实现自己的一个对象选择器。这样就可以限定其他人员只能在指定的路径来选择设置对象，不会导致设置错误的对象。另外，默认的对象选择器，只能普通预览对象，而不能像在检视器预览里那样进行操作，比如音频的播放，动画的播放预览等。

# 功能
* 可以设置指定路径
* 可以操作资源的播放预览
